  • Surawong’s
    Values
“昼は有名な観光地をめぐり、
夜はココテルを楽しむ”
ようこそ、小さなお子様がいるファミリーのみなさん。お子さん連れの旅行は大変と思い込んでいませんか？もしお子さんが疲れて早めにホテルに帰っても、ココテルには子供が喜ぶ仕掛けがたくさん。カフェ@ココテルでは子供向けの安心メニューもご提供。お子様がお休みになったあと、夫婦だけの時間を過ごすのもいいアイディアかもしれません。
  • Surawong’s
    Values
“この旅の本当の思い出は、
ホテルに帰ってから創られる”
ようこそ、大切な仲間がいる若者のみなさん。あとどれくらい、この仲間で一緒にいられるでしょうか。旅は気の置けない仲間の知らなかった一面を知る機会を与えてくれます。旅の思い出や将来の夢を夜まで一つ屋根の下で話し込むのにピッタリのお部屋、カフェ、レストランをココテルは提供します。この仲間が一生の宝物になりますように。
  • Surawong’s
    Values
“ココテルは活動的に動く
あなたの秘密基地”
ようこそ、日常社会で誰よりも頑張っているみなさん。たまには自分へのご褒美で、エネルギーと笑顔のあふれるバンコクを旅してみませんか。活動的なあなただからこそ、ゆっくり最高品質のベッドでお休みになってほしいとココテルは考えています。ここはあなたのセカンドハウスです。

KokoFamily（3人さま向け）

21平米のお部屋に3つのシングルベッドを配置した個性あふれるお部屋です。どこか日本の旅館を彷彿とさせるそのスタイルは、並んでお休みになりたいご家族、夜中まで話の尽きないお友達にピッタリです。

KokoParty（4人さま向け）

独創的なデザインが目を引く、特別ルームです。24平米のお部屋に4名さま分のベッドを配置しています。普段は同じ部屋に泊まれない4人家族、お友達グループにピッタリです。
